# Marine Engine Monitoring System

Sistema de monitoreo y análisis predictivo de motor marino simulado.

## Stack
- Python 3.11+
- pandas, numpy, plotly, streamlit
- AWS S3, Lambda, CloudWatch

## Setup
```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```
